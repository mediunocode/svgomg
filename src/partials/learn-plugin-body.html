What {{ plugin.name }} does in SVGO

SVGO applies {{ plugin.id }} as part of its plugin pipeline. On SVGOMG.net the toggle is labeled {{ plugin.name }} {% if plugin.enabledByDefault %}(enabled by default){% else %}(off by default){% endif %}. Optimization runs entirely in your browser — files are not uploaded to a server.

Use SVGOMG.net with Show original to compare before and after. If output looks wrong, disable this plugin first, then re-enable related transforms one at a time.
